Introduction to Python 

Python is a simple and powerful programming language used all over the world in science, web development, artificial intelligence, and robotics. Its great for learning because its commands look almost like normal English, so you can quickly understand what your code does.

With Python, we can make the computer calculate, draw, or collect data, and even explore space-related topics like predicting when and where a solar eclipse will happen. Learning Python helps you think like a scientist and a programmer at the same time.

In MakeCode, click the Python tab (at the top) to view and edit the program in Python.

1- Change the pause time to 1000 ms and test the program.
Observe how recording data every second affects the table and the graphs.

2- Change the temperature to be shown in Kelvin instead of Celsius.

Remember: Kelvin = C + 273.15.
